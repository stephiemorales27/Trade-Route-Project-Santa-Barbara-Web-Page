# Trade Route Project: Santa Barbara Web Page

## Overview

This is a comprehensive web application dedicated to showcasing the trade routes, attractions, and dining experiences of Santa Barbara. It integrates front‑end and back‑end components to deliver a responsive and user‑friendly interface. The application includes:

### Key Features

- **Restaurant Directory & Information** - Browse and explore restaurants in Santa Barbara with detailed information, elements, and dining options
- **Weather Integration** - Real-time weather information and forecasts for Santa Barbara to help plan your visit
- **QR Code Generator** - Generate and scan QR codes for easy access to location information and sharing
- **Contact Form** - User-friendly contact form with email integration for inquiries and feedback
- **Responsive Design** - Built with the Materialize CSS framework for a modern, mobile-friendly interface

### Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript, Materialize CSS Framework
- **Backend:** PHP
- **Email Service:** PHPMailer for secure email handling
- **Design Framework:** Materialize v0.97.7

### Installation & Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/stephiemorales27/Trade-Route-Project-Santa-Barbara-Web-Page.git
   ```

2. **Local Server Setup**
   - This project requires a local PHP server to run properly
   - Use XAMPP, WAMP, or MAMP to set up a local development environment

3. **Navigate to the Project**
   ```bash
   cd Trade-Route-Project-Santa-Barbara-Web-Page
   ```

4. **Start Your Local Server**
   - Place the project folder in your server's htdocs (XAMPP) or www (WAMP) directory
   - Start Apache and PHP modules
   - Access the application through `http://localhost/Trade-Route-Project-Santa-Barbara-Web-Page`

### Usage

- **Explore Restaurants** - Navigate through the restaurant section to discover dining options in Santa Barbara
- **Check Weather** - View current weather conditions before planning your activities
